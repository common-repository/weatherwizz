Puts your WeatherWizz dashboard data directly onto your wordpress website giving you control on how your data is displayed. 


== Description ==

Using simple shortcodes provided by the WeatherWizz plugin you can now feed your WeatherWizz dashboard weather 
data directly into your wordpress site. 

How to use?

Login to your WeatherWizz account and navigate to integration and select WordPress, from here you can create WordPress shortcuts to display on your webpages.

See examples below.


Today the temperature in my garden is {{wiz_abc00}}c

When the page is opened this will be displayed as:
Today the temperature in my garden is 18.7c.

You can even configure your sensors to display in multiple formats on the same page.

See more examples here https://www.weatherwizz.com/

== Frequently Asked Questions ==

= What weather stations can I use? =

Any weather station that is connected to WeatherWizz via meteobridge, weatherbridge, weather display software, cumulusMX, Ecowitt, Ambient, Fine Offset.

= How often is my data updated? =

Your weather data is updated automatically every 30 seconds. 

= Can I display data from multiple stations on the one page? =

Sure, just log into the WeatherWizz account associated with the weather station, navigate to integration and select WordPress, from here you can create WordPress shortcuts to display on your web pages.

The WeatherWizz WordPress plugin can support sensors from multiple WeatherWizz accounts.
